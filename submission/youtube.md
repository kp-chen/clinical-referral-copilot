## Title

Clinical Referral Copilot — OpenAI Build Week Demo

## Description

Clinical Referral Copilot is a clinician-in-the-loop referral triage prototype built for OpenAI Build Week 2026.

It turns a synthetic gastroenterology referral into a structured urgency recommendation with exact phrase-to-rationale traceability, strict Structured Outputs, visible uncertainty, and an explicit clinician approve/edit/reject decision.

Live demo: https://clinical-referral-copilot.vercel.app

Source: https://github.com/kp-chen/clinical-referral-copilot

Built with GPT-5.6, the OpenAI Responses API, Codex, Next.js, React, TypeScript, and Vercel.

The narration is AI-generated using OpenAI `gpt-4o-mini-tts` with the Cedar voice.

Synthetic demonstration only. Not a medical device. Not for clinical use. No real patient data is used.
